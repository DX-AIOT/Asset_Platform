# Backend API

NestJS backend API cho Asset Platform.

## Development

```bash
npm run dev
```

API sẽ chạy tại http://localhost:3001

## Tech Stack

- NestJS
- TypeScript
- PostgreSQL
- Redis
- OpenAI Integration
- JWT Authentication

## API Documentation

API docs sẽ có tại http://localhost:3001/api/docs (Swagger)
