# Web Dashboard

Next.js web dashboard cho Asset Platform.

## Development

```bash
npm run dev
```

App sẽ chạy tại http://localhost:3000

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- React Query (TBD)
